### 1. 技术栈 (Technology Stack)

这个项目采用了 **Monorepo** (多包管理) 结构，核心由两部分组成：

*   **核心解析库 (`packages/`):**
    *   作者完全使用 **TypeScript** 手写了 Epub, Mobi, AZW3, FB2 的解析器。
    *   **关键点：** 它不需要 Rust 或 Python 后端，直接在浏览器 JavaScript 环境中解析二进制电子书格式（Mobi/AZW3）。这是非常少见且高难度的实现。
*   **前端应用 (`reader-html/`):**
    *   **框架:** [Vue.js 3](https://vuejs.org/)
    *   **语言:** TypeScript
    *   **构建工具:** 看起来是 Vite 或 Rollup（现代 Vue 项目标配）。
    *   **样式:** SCSS/CSS
